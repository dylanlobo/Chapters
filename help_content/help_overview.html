Overview

File Operations
- Load, Save and Reload
YouTube Operations
- Load chapters directly from a YouTube URL (or video identifier)
Player Control Operations
- Connect/Disconnect from a running (MPRIS enabled) audio/video media player.
- Play/Pause
- Skip forwards or backwards by 5 seconds, 10 seconds or 1 minute
- Jump directly to specified time offset.
- Raise the the connected player's window.
Chaptering Operations
- Jump to a selected chapter
- Create/Edit the title of a video/audio playback.
- Create/Insert/Edit a chapter title with corresponding time offset
- Delete a selected chapter.
Theme Selection:
Changed the GUI theme/appearance by selecting from a list of supported GUI themes.
Keyboard Shortcuts:
Supports several keyboard shortcuts for the above mentioned operations.
See the help section on keyboard shortcuts for details.
Note:
Chaptering information can be stored and loaded from JSON files with a .ch extension. The general structure of a Chapters file is:
{ "title":"my title", "chapters":{ "Introduction":"00:00:00", "Chapter 1":"00:01:00" } }
"title" is the title of the audio/video file. "chapters" is a map collection of chapter titles and their corresponding time offsets in HH:MM:SS format.
